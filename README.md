Keyboard Layout Fixer is a browser extension designed to quickly correct text typed in the wrong keyboard layout.

It allows you to seamlessly convert text between keyboard layouts directly inside:

* Standard input fields
* textarea fields

* Most custom input components
  
You can convert either the entire text or just a selected fragment.

**Note:** Due to technical limitations, the extension may not work in some highly customized or protected input fields. If you encounter such a case, you can report the problematic field via the context menu.

## Features

* Convert selected text or full text between keyboard layouts

* Support for standard and most custom input fields

* Automatic layout correction system

* Auto-mode control via context menu:

* Enable / Disable automatic correction

* Exclude specific websites

## Technology

Written in JavaScript

Built using Manifest V3
